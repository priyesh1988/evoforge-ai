# Examples

## Example: Successful recommendation
- Behavior shows repeated abandonment in settings.
- AI identifies excessive complexity.
- Proposal recommends guided setup + progressive disclosure.
- Simulator shows reduced abandonment and ticket rate.
- Team approves an experiment.

## Example: Failure-aware rejection
- Data is skewed toward expert admins.
- AI suggests hiding advanced options globally.
- Simulator warns about regression for expert workflows.
- Confidence is downgraded.
- Team rejects the proposal until broader telemetry is available.
